# xClock - Windows Digital Clock

Electronで作成されたシンプルで高機能なデジタル時計アプリケーションです。

## 概要

xClockは、Windowsで動作するクロスプラットフォーム対応のデジタル時計です。現在の時刻表示に加え、複数の便利な機能を搭載しています。

## 主な機能

- **デジタル時計表示** - 大きく見やすい時刻表示
- **アラーム機能** - 指定時刻での通知
- **タイマー機能** - カウントダウン機能
- **メッセージ表示** - カスタムメッセージ
- **天気情報** - 天気データの表示
- **祝日表示** - 日本の祝日対応
- **音声設定** - 通知音の制御
- **フォント設定** - 表示フォントのカスタマイズ
- **テーマ設定** - カラーテーマの切り替え
- **インポート/エクスポート** - 設定の保存・復元

## 必要な環境

- Node.js 14.0以上
- npm または yarn
- Windows 7以上

## インストール

### 1. 開発環境のセットアップ

```bash
npm install
```

### 2. 開発モードで実行

```bash
npm start
```

### 3. アプリケーションのビルド

```bash
npm run build
```

ビルド後、アプリケーションはNSIS形式のインストーラーとして生成されます。

## 使用方法

アプリケーション起動後、以下の操作が可能です：

- **タブ切り替え** - 各機能タブ（アラーム、タイマー、メッセージ、天気予報）を選択
- **設定変更** - テーマ、フォント、サウンド設定を調整
- **データ管理** - 設定のインポート/エクスポート機能
